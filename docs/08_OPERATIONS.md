# 运维与发布

更新时间：2026-03-22

## 1. 每次发布标准步骤

1. 本地改代码
2. 本地校验
3. 提交并推送 `main`
4. 微信云托管从 `main` 发版
5. 用户端和商家端清缓存并重新编译

## 2. 每次发布后建议验证

### 用户端

- 登录
- 菜单页选菜
- 实时金额
- 购物车核对
- 下单
- 上传付款截图

### 商家端

- 登录
- 商品管理
- 规则管理
- 店铺设置
- 用户中心
- 订单审核

## 3. 当前常见问题

### 看见旧 UI

通常是微信开发者工具缓存。

处理方式：

- 清缓存并重新编译

### 云托管规则未生效

通常是云端还没拉到最新 `main`。

### 登录转圈

优先排查：

- `/api/user/login`
- 云托管日志
- 小程序请求域名
